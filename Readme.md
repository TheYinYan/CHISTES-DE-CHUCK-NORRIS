# [Chistes de Chuck Norris](https://theyinyan.github.io/CHISTES-DE-CHUCK-NORRIS/)

## Descripción del proyecto
Este proyecto es una aplicación web que muestra chistes de Chuck Norris utilizando la API de Chuck Norris. La aplicación permite a los usuarios seleccionar una categoría de chistes, obtener un chiste aleatorio y copiar el chiste al portapapeles.

## Características principales
- Selección de categorías de chistes.
- Obtención de chistes aleatorios.
- Copia de chistes al portapapeles.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Estructura del proyecto
- `index.html`: Archivo principal que contiene la estructura de la página web.
- `css/estilos.css`: Archivo de estilos para la apariencia de la página.
- `js/javascript.js`: Archivo de JavaScript que contiene la lógica de la aplicación.
- `README.md`: Archivo de documentación del proyecto.

## Cómo ejecutar el proyecto
### Localmente
1. Clona el repositorio o descarga los archivos del proyecto.
2. Abre el archivo `index.html` en tu navegador web.
3. Interactúa con la aplicación para obtener chistes de Chuck Norris.
